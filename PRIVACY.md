# プライバシーについて

本公開版にはログイン、広告、アクセス解析、利用データを外部送信する機能はありません。

所持枚数、直筆、欲しい、推し設定などはブラウザのlocalStorageに、利用者が設定したメンバー画像はIndexedDBに保存されます。これらは運営者へ送信されません。

ブラウザのサイトデータ削除、端末変更、URL変更などにより保存内容が消失する場合があります。定期的なバックアップを推奨します。画像は通常バックアップに含まれません。


## セキュリティ設計

- 読み込むJavaScript・通信先を同一サイト内に限定するContent Security Policyを設定しています。
- バックアップJSONはファイル容量・キー・値・登録対象を検証してから復元します。
- メンバー画像はJPEG・PNG・WebPの静止画像に限定し、SVGなどは受け付けません。
- 本サイトには外部送信API、広告SDK、アクセス解析SDKを組み込んでいません。

GitHub Pagesの仕様上、すべてのHTTPセキュリティヘッダーをサイト側から自由に設定できるわけではありません。
