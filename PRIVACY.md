# プライバシーについて

本公開版にはログイン、広告、独自のアクセス解析、所持情報・推し設定・設定画像を外部送信する機能はありません。

所持枚数、直筆、欲しい、推し設定などはブラウザのlocalStorageに、利用者が設定したメンバー画像はIndexedDBに保存されます。本サイト運営者は、これらのデータを収集または閲覧しません。

## GitHub Pagesによる配信

本サイトはGitHub Pagesを利用して配信しています。GitHubはサービスの運用・セキュリティ目的で、サイトへのアクセス時にIPアドレスなどの技術情報を記録・保存する場合があります。これは本サイト独自のアクセス解析ではありません。

## 端末内データ

ブラウザのサイトデータ削除、端末変更、URL変更などにより保存内容が消失する場合があります。定期的なバックアップを推奨します。メンバー画像は通常バックアップに含まれません。

共有端末での利用は避け、端末ロックをご利用ください。端末内に保存されるデータについて、暗号化保管を保証するものではありません。

## セキュリティ設計

- 読み込むJavaScript・通信先を同一サイト内に限定するContent Security Policyを設定しています。
- バックアップJSONはファイル容量・キー・値・登録対象を検証してから復元します。
- メンバー画像はJPEG・PNG・WebPの静止画像に限定し、SVGなどは受け付けません。
- 本サイトには外部送信API、広告SDK、アクセス解析SDKを組み込んでいません。
- 同じOrganization配下で別のGitHub Pagesサイトを公開しない運用とし、同一オリジンの範囲を増やさない方針です。

GitHub Pagesの仕様上、すべてのHTTPセキュリティヘッダーをサイト側から自由に設定できるわけではありません。
