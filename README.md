# ＝LOVE 生写真管理 Ver1.00

＝LOVEの生写真コレクションを、スマートフォンやPCのブラウザで管理できる非公式ツールです。

## 主な機能

- メンバー別・全メンバーの所持枚数管理
- ヨリ・チュウ・ヒキの登録
- 直筆・欲しい・提供可能の管理
- 未所持一覧、コンプ率、年代別統計
- 現役メンバー・卒業メンバー対応
- 最推し・推し・気になる設定
- 利用者の端末内だけで使えるメンバー画像設定
- JSONバックアップ・復元
- PWA・ホーム画面追加・オフライン利用

## 非公式ツールについて

本サイトは、＝LOVEのファンが個人で制作・運営する非公式の生写真管理ツールです。＝LOVE、所属事務所、レコード会社およびその他の関係各社とは一切関係ありません。

＝LOVEに関する名称、作品、画像、商標その他の権利は、それぞれの権利者に帰属します。本サイトには公式写真やメンバー画像を収録・配布していません。

## 画像と保存データ

利用者が設定したメンバー画像、所持枚数、直筆、欲しい、提供可能、推し設定などは、利用者の端末内に保存されます。本サイトはこれらを外部サーバーへ送信・収集しません。

ブラウザのサイトデータ削除、端末変更、公開URLの変更などによりデータが消失する場合があります。定期的にバックアップファイルを保存してください。メンバー画像は通常バックアップに含まれません。

## iPhoneでアプリとして使う

1. Safariで本サイトを開く
2. 共有ボタンを押す
3. 「ホーム画面に追加」を選ぶ
4. ホーム画面のアイコンから起動する

詳細はアプリ内の「本サイトについて・利用上の注意」、`PRIVACY.md`、`DISCLAIMER.md`をご確認ください。
