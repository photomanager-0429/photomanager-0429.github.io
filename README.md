# 【非公式】＝LOVE 生写真管理

＝LOVEの生写真について、所持枚数・直筆・欲しい・提供可能・未所持などを端末内で管理できる、ファン個人制作の非公式ツールです。

## 重要なご案内

本サイトは、＝LOVEのファンが個人で制作・運営する非公式ツールです。＝LOVE、所属事務所、レコード会社、運営会社およびその他の関係各社とは一切関係ありません。

＝LOVEに関する名称、作品、画像、商標その他の権利は、それぞれの権利者に帰属します。本サイトには公式写真やメンバー画像を収録・配布していません。

## 主な機能

- メンバー別・全メンバー横断の所持管理
- ヨリ・チュウ・ヒキ別の所持枚数、直筆、欲しい登録
- 未所持、提供可能、統計、コンプ率
- クイック入力、イベント別チェック表、一括操作
- 最推し・推し・気になる設定
- 利用者が選択したメンバー画像の端末内表示
- JSONによるバックアップ・復元
- iPhoneのホーム画面追加、オフライン表示

## 保存データとプライバシー

所持情報と推し設定はlocalStorage、設定画像はIndexedDBを使用し、利用者の端末内へ保存します。本サイトにはログイン、広告、独自のアクセス解析、所持情報や設定画像を外部送信する機能はありません。

本サイトはGitHub Pagesを利用して配信しており、GitHubはサービスの運用・セキュリティ目的でアクセス時のIPアドレスなどの技術情報を記録・保存する場合があります。本サイト運営者は、利用者の所持情報・推し設定・設定画像を収集または閲覧しません。

共有端末での利用は避け、端末ロックをご利用ください。端末内データについて、暗号化保管を保証するものではありません。詳細は `PRIVACY.md` を確認してください。

## 画像設定について

利用者が選択した画像は端末内だけに保存され、外部サーバーへ送信されません。JPEG・PNG・WebPのみを受け付け、SVGは拒否します。画像の権利と入手元をご確認のうえ、個人利用の範囲で使用してください。

## セキュリティ

CSP、外部通信制限、バックアップJSONの厳格検証、画像形式・容量・解像度制限、Service Workerの厳格なキャッシュ許可リストを設定しています。

脆弱性は公開Issueに詳細を書かず、GitHubのPrivate vulnerability reportingから報告してください。詳細は `SECURITY.md` を確認してください。
